## 🔧 自测发现并修复的缺陷
**现象**：加入 JWT 鉴权后，租户管理员点击开通邮件中的激活链接访问 `/api/activation/{token}` 返回 `401 需要登录`——但管理员此时尚无账号，激活流程被阻断。
**根因**：`JwtAuthFilter` 拦截所有 `/api/**`（仅放行 `/api/auth/login`），漏放行激活接口。
**修复**：过滤器放行 `/api/activation/**`（提交 `7f5a35f`）。
**回归用例**：`AuthIntegrationTest.activationEndpointIsPublic` —— 无 token 访问激活接口不得返回 401。
