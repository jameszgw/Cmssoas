功能说明:模板资产管理(CmPrint 云端模板库 + 审批流)

把 CmPrint 打印模板作为受管资产纳入平台:细粒度权限(按角色分配 查看/编辑/使用/删除/导出/审批)、 变更审批流(送审中只读,审批通过方生效)、版本留档/回滚,并以按租户的模板库密钥向 CmPrint 设计器分发(企业版能力 templateGallery 的平台侧落地)。沿用平台一贯设计:RBAC、审计、Flyway。

审批流(状态机)

DRAFT(草稿) ──送审──> PENDING(审批中,模板只读) ──通过──> APPROVED(版本生效,进模板库)
                         │                                │
                         └──驳回──> 回到可编辑(有生效版回 APPROVED,否则回 DRAFT;原生效版不变)
APPROVED ──下架──> DISABLED(退出模板库,数据保留) ──上架──> APPROVED
  • 编辑只改草稿(draft_content),生效内容(content/current_version)只能经审批写入;
  • 每次送审留档一个版本(提交人/备注/审批人/意见/内容哈希),驳回也留痕——轨迹永不丢失;
  • 回滚 = 把历史版本内容拉回草稿,再走送审→审批生效(避免绕过审批的"直接回滚");
  • 全部动作写审计(`TPL_CREATE/UPDATE/SUBMIT/APPROVE/REJECT/ROLLBACK/DELETE/DISABLE/ENABLE/EXPORT/

GALLERY_SAVE/GALLERY_KEY_RESET`),已纳入「CmPrint 授权」页的审计查询范围。

细粒度权限(V20 权限点,按角色分配)

权限点控制
tpl:view查看列表/详情/版本
tpl:edit新建/改草稿/送审/回滚/上下架/模板库密钥管理
tpl:approve审批通过/驳回(与编辑分权,可两人复核)
tpl:use使用(预留给按模板调用打印的宿主集成)
tpl:delete删除(含全部版本)
tpl:export导出模板 JSON(每次导出记审计:谁导出了什么)

无权限的按钮在控制台直接隐藏(auth.has),后端 @RequirePerm 强制。

云端模板库(CmPrint 设计器直连)

  • 运营端「模板资产 → 模板库密钥」按租户生成密钥(PUBLIC=平台公共库),得到形如

https://平台地址/pub/cmprint/gallery/{galleryKey} 的 cloudBaseUrl;

  • CmPrint 侧只需 <cmprint-designer :cloud-base-url="上述地址" :cloud-owner="用户名">,

设计器「模板库」即可列出/载入云端模板(契约 = cmprint cloud-api.js:

GET|POST /api/templates、POST /api/templates/{id}/use、DELETE /api/templates/{id}?owner=, 响应 {success, message?, data});

  • 可见范围:租户密钥 = 平台公共 + 本租户的 APPROVED 模板;PUBLIC 密钥仅平台公共;
  • 设计器上传落为 PENDING(运营审批通过才进模板库)——上传共享不绕过审批;
  • 属主只能删除本租户尚未生效的上传;已生效模板的下架/删除走运营端;
  • 密钥可重置(旧地址立即失效);use 计数回流到资产列表(使用热度)。

API

运营端(/api/tpl/**,JWT+RBAC):列表/详情/版本 GET /api/tpl[,/{code},/{code}/versions]、 创建 POST /api/tpl、改草稿 PUT /api/tpl/{code}、送审/通过/驳回

POST /api/tpl/{code}/submit|approve|reject、回滚 POST /api/tpl/{code}/rollback/{version}、

上下架 POST /api/tpl/{code}/enable|disable、删除 DELETE /api/tpl/{code}、 导出 GET /api/tpl/{code}/export、密钥 GET /api/tpl/keys + POST /api/tpl/keys/{tenant}[/reset]。

公开端(模板库密钥鉴权):/pub/cmprint/gallery/{key}/api/templates...(见上节契约)。

数据(V20 迁移)

print_template(主表:生效内容+草稿+使用计数)、print_template_version(版本轨迹)、 template_gallery_key(按租户密钥);MySQL 方言同步;权限种子 6 点(SUPER_ADMIN 自动 FULL)。
