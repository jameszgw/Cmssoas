Get Started

5 分钟，从部署到应用

默认 H2 开箱即跑，初始账号 admin / 8888；生产一键容器化。

1

启动后端

默认 H2，无需安装数据库，一条命令拉起服务。

2

启动前端

Vue3 控制台 npm 安装即跑，浏览器登录运营台。

3

开通与签发

开通租户、配置产品套餐、签发 License、上传 jar 加固。

4

容器化上线

复制 .env 模板，docker compose 一键部署生产环境。

# 后端（默认 H2，开箱即跑；国密模式加 LICENSE_SIGN_ALGO=sm2）
cd server/license-platform && mvn spring-boot:run        # http://localhost:8080

# 前端（初始账号 admin / 8888）
cd web/console && npm install && npm run dev             # http://localhost:5173

# 生产一键容器化
cp .env.example .env        # 试用；或 cp .env.prod.example .env（生产）
docker compose up -d --build
