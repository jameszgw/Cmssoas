# CODEMAN 智能客服知识库(FAQ)

> 客服仅依据本知识库与产品须知回答“本产品相关”问题；命中条目作为参考资料注入提示。
> 每个条目以 "## " 开头为问题，其下正文为答案，空行分隔条目。

## CODEMAN 是什么
CODEMAN 是软件授权运营平台，提供代码保护、License 授权认证、在线授权(SDK 通道)、产品与套餐订阅、计费开票、多租户运营等能力。后端 Spring Boot + 前端 Vue3 控制台。

## 如何登录运营控制台
访问控制台地址，使用运营账号登录。初始超级管理员账号为 admin，首次登录后请按提示修改密码。账号与角色权限由“角色权限/用户管理”维护。

## 忘记密码或需要重置怎么办
请联系超级管理员，在“用户管理”中对账号执行“重置密码”，重置后用户首次登录会被强制修改密码。

## License 授权是怎么签发的
在“License 授权”页创建授权：选择产品/套餐、绑定客户与到期时间，系统用服务端私钥签名生成 .lic 授权文件供客户使用。支持 Ed25519(默认)与国密 SM2 两种签名算法。

## License 过期或临期提醒
系统会对临近到期的 License 发送邮件提醒(发件箱异步发送)。可在“License 授权”页查看状态(生效/临期/过期/已吊销)，并支持导出 CSV。

## 在线授权(SDK 通道)如何工作
客户端集成 License SDK，通过在线通道激活与心跳续期；断网有宽限期，超时席位释放。SDK 通道有每 IP 每分钟限流，防滥用。

## 套餐订阅与计费
在“产品与版本/套餐订阅”维护产品与套餐；订阅或套餐变更会自动出账(账单 PENDING)。在“计费账单”中先“收款”再“开票”生成发票号。金额=套餐价×数量。

## 多租户隔离
平台按 Schema-per-tenant 物理隔离各租户数据，互不可见。租户开通后自动初始化其 Schema 与角色。

## 用户须知与授权(同意)
“须知与授权”集中管理服务条款/隐私政策/公告，版本化发布；强制确认的须知会在登录后弹层阻断直至用户确认。所有同意/撤回记录 IP/UA/版本/时间，可查询导出供取证。

## 合同签约(电子签)
可基于订阅生成合同并发起多方签署；各方签署完成后对合同内容做哈希存证归档。支持自建签章留痕；如需更强法律效力可对接第三方合规电子签。

## 数据安全与密钥管理
私钥不落明文，经环境变量/Nacos/KMS 注入。智能客服不会外发私钥、密码、完整密钥串或敏感个人信息。可配置本地 Ollama 模型实现完全内网离线。

## 转人工客服
若问题智能客服无法解决，可在对话窗点击“转人工”，会话状态转为已转人工并留存记录，由人工跟进。
