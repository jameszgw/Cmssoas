# CODEMAN 帮助文档 / FAQ

## 账号与登录
**Q：初始账号是什么?** admin / 8888,首次登录强制改密。
**Q：忘记密码?** 由超管在「用户管理」重置;重置后用户首登强制改密。
**Q：登录后被弹层挡住?** 存在「强制确认」的须知,勾选「我已阅读并同意」即可继续。

## License 授权
**Q：.lic 如何生成、如何防伪?** 服务端私钥签名(Ed25519 默认 / 国密 SM2),客户端 SDK 用公钥验签;私钥不出域、不落明文(env/Nacos/KMS 注入)。
**Q：吊销后离线客户端如何感知?** 拉取公开签名 CRL（`/pub/crl`)并用公钥(按 kid)验签后拒绝已吊销 License;公钥集见 `/pub/license/public-keys`。
**Q：到期会自动停用吗?** 会,系统每日自动将过期 License 置为 EXPIRED;也可在「License 授权 → 到期停用」手动触发。在线通道实时拒绝吊销/过期。

## 计费 / 支付 / 发票
**Q：怎么收款?** 「计费账单 → 收款」发起扫码;客户支付后财务核对到账,点「确认到账(人工)」即收款。默认人工确认,无需对接网关;如需自动到账可接微信支付/支付宝/Stripe(改 `app.pay.provider`)。
**Q：能开正规发票吗?** 能。已收款账单「开票」,填票种(普票/专票)、抬头、税号、邮箱,生成发票代码/号码/PDF。默认沙箱;生产改 `app.einvoice.provider` 接航信/百望/税务平台。

## 智能客服(大模型)
**Q：必须联网/买 API 吗?** 不必须。未接入大模型时自动降级为知识库匹配;要更强可接**本地 Ollama(离线免费)**或云 API(智谱 GLM-4-Flash 常免费、通义 qwen-turbo 极低价、DeepSeek 等)。
**Q：怎么换模型?** 只改 `.env` 三项:`AI_BASE_URL / AI_API_KEY / AI_MODEL`,代码不动。详见 `docs/智能客服-大模型选型与部署.md`。
**Q：会泄露敏感信息吗?** 系统提示限定仅答本产品、不外发私钥/密码/隐私;配本地 Ollama 可全程内网离线。

## 代码加固
**Q：是上传 jar 后自动混淆吗?** 提供两种并存方式:① 构建/打包时(ProGuard+类加密,见示例工程);② 平台在线(「代码加固」页上传 jar→异步加固→下载)。是否启用在线由租户「加固模式」决定(BUILD/ONLINE/BOTH)。
**Q：在线加固支持哪些?** ProGuard 混淆、关键类加密+License 绑定、口令式类加密;可叠加。
**Q：加固后怎么运行?** License 绑定 `java -Dharden.license=<.lic> -jar`;口令式 `java -Dharden.key=<口令> -jar`。缺/错密钥则无法解密运行。
**Q：能接商用混淆器吗?** 可。HardenProvider 抽象,接 Allatori/ClassFinal/Xjar 只需新增实现。

## 租户自助门户
**Q：客户怎么自助查看?** 运营在「自助门户」为其开通并给出「租户编号 + 访问码」;客户在门户地址登录,只读看本租户数据,数据严格隔离。
**Q：访问码泄露怎么办?** 「重置」生成新码,旧码立即失效;也可「停用」门户。

## 部署 / 环境
**Q：支持哪些数据库?** PostgreSQL(生产推荐)、MySQL 5.7/8.0、H2(开发)。MySQL 库需 utf8mb4。迁移由 CI 真机验证。
**Q：支持 Java 8 吗?** 不支持。需 **JDK 17+(推荐 21)**(Spring Boot 3.5 要求)。
**Q：怎么初始化数据库?** 推荐让应用首启自动建表(Flyway);或用 `deploy/sql` 纯 SQL 初始化(schema → init-admin → test-data)。
**Q：怎么改密钥/数据库/邮件等配置?** 用 `.env`(参考 `.env.example` / `.env.prod.example`);生产务必改 `JWT_SECRET` 与数据库口令。

## 常见排错
- **登录 401/页面空白**:检查后端是否就绪(`/actuator/health`)、JWT 密钥一致、前端 `/api` 反代正确。
- **中文乱码(MySQL)**:库与表须 utf8mb4,连接串用 `characterEncoding=UTF-8`。
- **MySQL 启动报 “No database found to handle jdbc:mysql”**:确认含 `flyway-mysql` 依赖(本项目已内置)。
- **邮件未发送**:`APP_MAIL_DELIVERY=log` 仅落盘;真实外发设为 `smtp` 并配 `MAIL_*`。
- **智能客服回复“暂不可用”**:`AI_ENABLED=true` 但上游不可达;检查 `AI_BASE_URL`/网络/本地 Ollama 是否启动,或关掉以走知识库降级。
