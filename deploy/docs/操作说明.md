# CMSSOAS 功能操作说明

运营控制台按模块操作指引(每页内置 💡 操作指引与浅色功能简介)。初始账号 **admin / 8888**,首登请改密。
> 各功能的设计与接口详见仓库根 `docs/功能-*.md`。

## 登录与权限
- 浏览器打开控制台 → 用运营账号登录;首次或被重置后强制改密。
- 菜单按角色权限显示。角色与授权在「角色权限 / 用户管理」维护(超管可见)。
- 强制确认的「须知」会在登录后弹层,需勾选同意方可继续。

## 运营总览
- 查看租户数、有效 License、临期/告警、订阅与营收概览。

## 租户管理
- 「开通租户」填写名称/管理员邮箱/套餐 → 系统建租户独立 Schema 并发开通邮件(发件箱异步)。
- 管理员凭邮件链接激活、设密码、(可选)绑定 MFA。
- 可续期、查看租户隔离配置。

## License 授权
- 「签发 License」:选产品/版型/模块/有效期/并发 → 服务端私钥签名生成 `.lic`(Ed25519 或国密 SM2)。
- 续期、变更、吊销;下载 `.lic`;查看版本历史 diff;导出 CSV。
- 「吊销名单(CRL)」:查看已签名 CRL 与公开分发地址(离线客户端用公钥按 kid 验签)。
- 「到期停用」:手动触发到期 License 批量停用(系统亦每日自动执行)。

## 在线监控
- 查看 SDK 在线实例、心跳、席位占用与宽限期释放。

## 产品与版本 / 套餐订阅
- 维护产品矩阵与套餐(价格/席位/模块/功能点/版本范围)。
- 创建订阅 → 自动出账(账单 PENDING),可联动签发。

## 客户管理(客户360)
- 维护客户主数据(联系人/邮箱/行业/关联租户)。
- 点客户「客户360」:按客户名称聚合其 License/合同/账单/支付/订阅与关键指标。

## 计费账单(收款 + 开票)
- 订阅/合同自动出账。
- 「收款」:发起扫码收款 → 客户支付后财务核对到账,点「确认到账(人工)」完成(默认人工确认;可接微信/支付宝/Stripe)。
- 「开票」:对已收款账单开具**正规电子发票**(普票/专票、抬头、税号、邮箱)→ 生成发票代码/号码/PDF;账单转「已开票」。
- 导出 CSV。

## 合同签约
- 选模板(占位符自动填充)或手填内容 → 新建草稿。
- 「发起签署」:内容定稿快照 + SHA-256 存证;各签署方逐一「签署」;全部签署后合同生效并存证归档,金额>0 自动出账。
- 可作废未签署合同;导出 CSV。

## 须知与授权
- 「须知管理」:新建草稿(条款/隐私/公告)→ 发布(内容定稿、旧版归档);可设「强制确认」。
- 「授权记录」:查看同意/撤回(主体/版本/渠道/IP/时间),导出 CSV 取证。

## 智能客服
- 控制台右下角悬浮窗对话;基于产品知识库(FAQ)作答,支持流式与「转人工」。
- 通用、不绑定厂商:后端 OpenAI 兼容接口,未接入大模型时降级为知识库匹配。
- 本页可审计全部会话与消息。

## 租户自助门户(面向最终客户)
- 运营在「自助门户」为租户「开通」生成访问码,「复制」凭证发给客户。
- 客户访问门户地址,用「租户编号 + 访问码」登录,**只读**查看本租户的 License/合同/账单/订阅(严格隔离)。

## 角色权限 / 用户管理
- 角色权限树支持「无/只读/编辑/完全」多态;新增权限自动授予超管。
- 用户创建/重置密码/启停;新用户首登强制改密。
